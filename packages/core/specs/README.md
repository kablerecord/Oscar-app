# OSQR Specifications

This folder contains component specifications for OSQR implementation.

## Adding Specs

1. Kable reformats specs in Web Claude using the Reformatter Prompt
2. Save output as `[component-name]-v1.md`
3. Update `OSQR-ROADMAP.md` to mark spec as added

## Format

All specs follow the standardized format with:
- Metadata (version, dependencies, status)
- Executive Summary
- Scope (in/out)
- Architecture
- Implementation Checklist
- API Contracts
- Success Criteria
- Open Questions
